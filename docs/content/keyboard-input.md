# Keyboard Input

Convert [`<code>`] to [`<kbd>`].

**Example**

``` markdown
Press `Control` + `J` to open the _Console_.
```

``` html
<p>Press <kbd>Control</kbd> + <kbd>J</kbd> to open the <em>Console</em>.</p>
```
