# Video

## Video

Convert video links to videos.

**Example**

``` markdown
[Video](demo.webm)
```

## YouTube

Convert YouTube video links to embedded videos.

**Example**

``` markdown
<https://youtu.be/qtIqKaDlqXo>
```

## Dailymotion

Convert Dailymotion video links to embedded videos.

**Example**

``` markdown
<https://dai.ly/x8nax4>
```
